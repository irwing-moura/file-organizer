
# File Organizer

Esta é uma simples aplicação responsável por facilitar a organização de arquivos em pastas.

![interface](https://raw.githubusercontent.com/irwing-moura/file-organizer/master/images/print1.png)

![interface](https://raw.githubusercontent.com/irwing-moura/file-organizer/master/images/print2.png)


## Requisitos

Para construir e executar o aplicativo, você precisa:

- [JDK 8](https://www.oracle.com/br/java/technologies/javase/javase8-archive-downloads.html)


## Como utilizar

- Execute a aplicação
- Clique no botão e escolha a pasta que deseja organizar
